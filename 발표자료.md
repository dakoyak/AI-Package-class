# AI 창의융합 교실 (AI Creative Classroom)

## 📌 프로젝트 개요

### 왜 만들었나?
초등학생들이 AI 기술을 직접 체험하며 창의력, 문제해결력, 협업능력을 자연스럽게 키울 수 있는 교육 플랫폼이 필요했습니다.

### 어떻게 활용하나?
- 교사는 학급별 맞춤 수업을 설계하고 학생 활동을 실시간 모니터링
- 학생은 게임처럼 재미있는 AI 체험을 통해 자기주도 학습
- 학교 현장에서 즉시 활용 가능한 올인원 교육 솔루션

### 기대 효과
✅ AI 리터러시 향상 - 윤리적 AI 사용법 체득  
✅ 창의적 사고력 증진 - 다양한 관점에서 문제 접근  
✅ 협업 능력 강화 - 토론과 관점 전환 훈련  
✅ 자기주도 학습 습관 형성 - 흥미 기반 능동적 참여

---

## 🎯 핵심 기능

### 1. 창의력 스튜디오
**AI 스파링** - 엉뚱한 반론으로 이야기 구조를 재구성하며 창의적 사고 확장  
**아트 워크숍** - 학생 그림을 명작 스타일로 즉시 변환하여 예술적 감각 개발  
**글쓰기 듀오** - 학년과 장르에 맞춘 AI 글쓰기 가이드로 표현력 향상

### 2. AI 리터러시
**편향 탐지 미션** - AI의 편향된 답변을 찾아내며 비판적 사고력 훈련  
**가드레일 챌린지** - 부적절한 AI 응답을 판별하며 윤리적 AI 사용법 학습

### 3. 몰입형 체험
**세종대왕 인터뷰** - RAG 기술로 구현된 역사 인물과 실시간 대화하며 역사 학습  
**AI 피트니스 코치** - MediaPipe 자세 감지로 운동 자세를 분석하고 실시간 피드백

### 4. 논리/협업
**스마트 토론** - 음성 인식으로 토론 내용을 분석하고 AI가 갈등 중재  
**관점 전환기** - 상대방 입장에서 상황을 재해석하며 공감 능력 향상

### 5. 학급 관리
**활동 기록** - 학생별 학습 활동 자동 추적 및 시각화  
**학급 게시판** - 사진과 글로 학급 소식 공유 및 소통  
**교사 관리** - 학생 등록, 공지사항 관리, 수업 자료 제작

## 🏗️ 시스템 아키텍처

### Frontend
```
React + TypeScript + Vite
├─ Styled-components (UI 스타일링)
├─ React Router (페이지 라우팅)
├─ Gemini AI (창의력, 리터러시 기능)
└─ MediaPipe (자세 감지)
```

### Backend
```
Node.js + Express
├─ SQLite (사용자/활동 데이터)
├─ Gemini AI (세종 인터뷰, 관점 전환)
├─ OpenAI (가드레일 검증)
├─ Multer (이미지 업로드)
└─ Edge-TTS (음성 합성)
```

### 데이터 흐름
```
학생/교사 → React UI → Express API → AI 모델 → 응답 처리 → UI 업데이트
                    ↓
                SQLite DB (활동 기록 저장)
```

### 주요 기술 특징
- **RAG (Retrieval-Augmented Generation)**: 세종대왕 지식 베이스 기반 정확한 역사 답변
- **실시간 자세 감지**: MediaPipe로 웹캠 영상 분석 및 즉각 피드백
- **멀티모달 AI**: 텍스트, 이미지, 음성을 통합 처리하는 Gemini 활용
- **안전한 인증**: bcrypt 해싱으로 학생/교사 계정 보안 관리
